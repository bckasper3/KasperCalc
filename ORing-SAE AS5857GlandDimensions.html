O-Ring Gland Design (SAE AS5857)

This tool references and improves the functionality of Design Table 4-1 from the Parker O-Ring Handbook (“ORD 5700”), covering gland design dimensions for O-ring and other elastomeric seals per SAE AS5857. It spans the full AS568 dash-number range (0XX, 1XX, 2XX, 3XX and 4XX series).

Use the search box to filter by dash number. All dimensions are in inches. Each dash number lists maximum and minimum limits for every gland feature.

Cross-section series: The first digit of the dash number encodes the nominal O-ring cross-section diameter (cs):

  • 2XX — 0.103″ cs, G† = 0.140″
  • 3XX — 0.139″ cs, G† = 0.187″
  • 4XX — 0.210″ cs, G† = 0.281″
  • 5XX — 0.275″ cs, G† = 0.375″

Remaining digits encode the O-ring inner diameter in a systematic sequence. The full dash number (e.g., 3-214) is the standard way to specify a Parker O-ring for procurement.
